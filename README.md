#  Dashboard de Vendas e Rentabilidade – Coca-Cola

##  Sobre o Projeto
Este projeto consiste na construção de um dashboard em Excel para análise de desempenho de vendas e rentabilidade, utilizando um dataset público disponível no Kaggle.

O objetivo foi explorar como vendas, lucro e eficiência operacional se comportam ao longo do tempo, regiões e produtos.

---

##  Objetivos da Análise
- Analisar o volume de vendas por região
- Avaliar a rentabilidade (Operating Profit e Margin)
- Identificar padrões sazonais nas vendas
- Comparar performance entre produtos
- Entender a eficiência operacional entre regiões

---

## Principais Insights
- A região **West** lidera em vendas e lucro, sendo a principal responsável pelo resultado geral
- Existem diferenças de eficiência entre regiões (nem todas convertem vendas em lucro da mesma forma)
- Foi identificado um padrão de sazonalidade, com picos de vendas ao longo do ano
- As vendas estão concentradas em poucos produtos principais (Coca-Cola, Dasani Water, Diet Coke)
- A margem operacional média (~36%) indica um bom nível de rentabilidade
